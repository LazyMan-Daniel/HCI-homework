Daniel's BEER PONG Game

Author/ designer:

  • Cristea Daniel Dumitru | cristea3000@yahoo.com

1. Abstract

The Daniel's BEER PONG will be the perfect app for people that love to party and drink. This app is based on the classic game of "Beer pong" that can be played in real life only if the people have: a big table, disposable cups, ping pong balls and alcohol. The app will have 2 modes: singleplayer and multiplayer.

2. Introduction - the purpose of the project

The app was build based on the notions learned from "Human-computer interface" class. Every element was chosen and positioned in such a way that the navigation through the app would be easy and fast. In the process of choosing every button of the interface i have asked at least 5 friends which button do they consider to be the best to suggest the functionality of them.

3. Overview

3.1 App functionality

  • Main page, where we have: game mode, settings, sound, store, access to facebook/instagram
  • Settings page, that includes: music on/off, sound on/off and touch sensibility
  • Gameplay page, where we have the game table with: a ball, 3 cups, points, coins, settings, no adds button, table and balls buttons

3.2 Type of users - Personas

To play this game an user doesn't have to authenticate because everything is stored localy in the phone memory. For single player mode the user will play with an AI and for multiplayer games, the player will use an unique code to get into an online room.

3.3 Operating enviroment

To play this game you will have to install it from App store or from Google Play. One installed you can play offline in singleplayer mode, but if you want to play multiplayer with your friends you will need a network connection.

3.4 User documentation + Wireframe

I. The main page - "Home page", that includes:

  • On the right side: buttons for Instagram and Facebook
  • In the middle: singleplayer and multiplayer buttons
  • At the bottom: settings button, music on/off and shopping cart

II. The settings page, that includes:

  • In the middle: sound ON and OFF, music ON and OFF and a sliding line for touch sensitivity

III. The game page, that includes:

  • UP - left : sound ON/OFF, store for tables, store for ping pong balls
  • UP - middle : number of points
  • UP - right : number of coins owned by the player, no adds button, buy more coins button
  • Middle : 3 cups of beer
  • DOWN : the ping pong ball

4. Interface description

4.1 Design

The game was designed for people 18+, because it is based on drinking alcohol in real life. The predominant colors used are: blue, red, green and pink with the purpose of making the game more attractive to the player. Every button in this game is based on icons that are used in IOS and Android apps, so that the player will already be familiarised with them.
